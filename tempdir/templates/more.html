ABOUT

A mobile application that can classify the breeding of the rabbit and give a percentage of what breed the rabbit is using a deep learnign model.

Specific Objectives:

-To design an application that will accuretly detect the correct breed of a rabbit.
-To design an application that will notify the user about a rabbits breeding status.
-To design an application that stores information about each rabbit that can be accessed by scanning the QR code and would provide important analytics to the user.
-To design and ecaluate the accuracy and reliabilty of the application.
